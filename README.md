# EMOBOT – Emotion-Aware AI Robot

This project is an emotion-aware robot that detects and responds to human emotions in real time using facial and voice analysis.

## Features
- Detects emotions using facial expressions
- Analyzes voice input for emotion recognition
- Provides real-time responses
- Integrates hardware and software components

## Technologies Used
- Python
- OpenCV
- NumPy
- Raspberry Pi
- Sensors and Motors

## Description
The system uses a camera and microphone to capture user input and applies basic machine learning techniques to identify emotions such as happiness, sadness, anger, and surprise. Based on the detected emotion, the robot responds using predefined actions.

## Applications
- Healthcare support
- Education assistance
- Emotional companion systems
